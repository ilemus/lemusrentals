### How to use

1. Clone the repository
2. Install dependencies `npm install`
3. Run the development server `npm run dev`

### Features

- Next.js 14 with app router
- Prebuilt components for a quick start
- Tailwind CSS
- Next Themes for dark mode
- React Icons
